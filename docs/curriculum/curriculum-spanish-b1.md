# Spanish Learning Curriculum - Level B1

Curriculum based on CEFR guidelines, Instituto Cervantes Plan Curricular (PCIC), and DELE B1 exam requirements.

## B1 Level Description

A person at B1 level (according to official CEFR guidelines):
- Understands the main points of clear standard input on familiar matters regularly encountered in work, school, leisure
- Can deal with most situations likely to arise while traveling in an area where the language is spoken
- Can produce simple connected text on topics that are familiar or of personal interest
- Can describe experiences and events, dreams, hopes and ambitions
- Can briefly give reasons and explanations for opinions and plans

B1 is the "threshold" level - the level of language independence.

## B1 Grammar

### Past Tenses - Pretérito Indefinido vs Imperfecto
- Pretérito indefinido (completed actions)
- Pretérito imperfecto (descriptions, habits, ongoing)
- Combining both in narration
- Time markers for each

### Present Subjunctive (Presente de Subjuntivo)
- Regular and irregular conjugation
- WEIRDO triggers (Wishes, Emotions, Impersonal, Recommendations, Doubt, Ojalá)
- Subjunctive vs indicative distinction
- Common expressions requiring subjunctive

### Imperfect Subjunctive (Pretérito Imperfecto de Subjuntivo)
- -ra and -se forms
- Use with conditional (Si clauses Type 2)
- Use after past tense main clauses
- "Como si" constructions

### Conditional Tense (Condicional Simple)
- Regular and irregular forms
- Polite requests (Me gustaría, ¿Podría...?)
- Hypothetical situations
- Advice (Yo en tu lugar...)

### Si Clauses (Oraciones Condicionales)
- Type 1: Si + presente, futuro/presente
- Type 2: Si + imperfecto subjuntivo, condicional
- Mixed conditionals
- Common expressions

### Relative Clauses (Oraciones de Relativo)
- Que, quien, donde, cuando
- El que, la que, los que, las que
- Lo que (what/that which)
- Preposition + relative pronoun

### Reported Speech (Estilo Indirecto)
- Verb tense changes
- Time and place adjustments
- Reporting questions
- Reporting commands

### Ser vs Estar - Advanced Uses
- With adjectives (different meanings)
- Passive voice (ser + participio)
- Resultant state (estar + participio)
- Expressions and idioms

### Object Pronouns - Combined
- Direct + indirect pronoun order
- Se lo, se la, se los, se las
- Position with infinitives and gerunds
- Redundant pronouns

### Por vs Para
- Cause vs purpose
- Duration vs deadline
- Exchange vs recipient
- Fixed expressions

### Future and Future Perfect
- Futuro simple (plans and predictions)
- Futuro perfecto (future perfect)
- Probability in present/past
- Ir a + infinitive vs futuro

### Commands (Imperativo)
- Affirmative tú and vosotros
- Negative commands (subjunctive)
- Usted/ustedes commands
- Pronoun placement

## Grammar Mapping to Topics

Grammar is integrated with topics - each set contains phrases using specific grammatical constructions.

| Topic | Grammar Constructions to Use |
|-------|------------------------------|
| **Travel - planning and reservations** | Condicional (Me gustaría, ¿Podría...?), Futuro (plans), Si clauses Type 1 |
| **Travel - transport and connections** | Cuando + subjuntivo, antes de que, después de que |
| **Travel - problems and delays** | Porque/como (cause), Condicional (polite complaints), Pasiva (El vuelo fue cancelado) |
| **Accommodation - hotel** | Condicional (requests), Relativos (room description), Para que + subjuntivo (purpose) |
| **Accommodation - problems** | Porque (explanations), Condicional (polite complaints), Pasiva (La habitación no ha sido limpiada) |
| **Travel stories** | Indefinido vs Imperfecto (narration), Relativos (place descriptions), Cuando + indicativo (past) |
| **Job hunting - CV** | Indefinido (experience), Después de + infinitivo (sequence), Participio as adjective |
| **Job interview** | Condicional (politeness), Porque/ya que (justifications), Futuro (career plans) |
| **At work** | Para que/para + infinitivo (purpose), Condicional (requests), Pasiva (processes) |
| **Work - problems** | Aunque (contrast), Porque (explanations), Condicional (suggestions) |
| **At the doctor - visit** | Condicional (polite questions), Desde hace/desde que (duration), Si + presente (conditional symptoms) |
| **Pharmacy - medicine** | Imperativo (instructions), Antes de/después de (medication sequence), Condicional |
| **Healthy lifestyle** | Para + infinitivo (purpose), Porque (justifications), Condicional (advice) |
| **Accidents and emergencies** | Indefinido (event description), Pasiva (what happened), Cuando + indefinido (sequence) |
| **Apartment hunting** | Relativos (apartment description), Condicional (questions), Por/debido a (because of location) |
| **Furnishing** | Pasiva (work), Para que (purpose), Participio as adjective |
| **Neighborhood** | Aunque (conflicts), Porque (explanations), Condicional (requests) |
| **Shopping - complaints** | Porque (problem), Condicional (polite demands), Pasiva (what happened to product) |
| **Services** | Condicional (requests), Para que/para (purpose), Mientras (during) |
| **Education** | Después de que (sequence), Porque/ya que (justifications), Futuro (plans) |
| **Language learning** | Aunque (difficulties), Para + infinitivo (learning purpose), Si clauses (success conditions) |
| **Technology - problems** | Pasiva (fault description), Porque (cause), Condicional (help request) |
| **Internet and social media** | Aunque (pros and cons), Porque (opinion justifications), Relativos |
| **Environment - ecology** | Porque/ya que (causes), Para que (action purposes), Pasiva (what is being done) |
| **Climate and weather** | Si clauses (conditions), Futuro (forecasts), Porque (phenomenon causes) |
| **Family - relationships** | Imperfecto (memories), Aunque (conflicts), Subjuntivo (wishes: Espero que) |
| **Friendship** | Desde que/desde hace (since when), Aunque (problems), Condicional (apologies) |
| **Feelings and emotions** | Porque (emotion causes), Aunque (contrast), Subjuntivo (wishes: Ojalá) |
| **Apologizing and conflicts** | Imperfecto subjuntivo (Si hubiera sabido), Porque (explanations), Aunque (although) |
| **Society** | Pasiva (what is being done), Porque/ya que (justifications), Para que (purposes) |
| **Politics** | Pasiva (political processes), Porque (justifications), Relativos (descriptions) |
| **Culture - entertainment** | Relativos (reviews), Porque (opinions), Indefinido (event memories) |
| **Hobbies and free time** | Desde que/desde hace (since when), Porque (why I like), Cuando + presente (when I practice) |
| **Future plans** | Futuro, Condicional (dreams), Si clauses (conditions), Para que (purposes) |
| **Life turning points** | Indefinido + Imperfecto (sequence), Porque (decision causes), Si + imperfecto subjuntivo (what if) |

## Differences Between A2 and B1

| Aspect | A2 | B1 |
|--------|----|----|
| Sentences | Simple, short | Complex, with conjunctions |
| Topics | Concrete, everyday | Abstract (opinions, plans) |
| Grammar | Basic tenses | Subjunctive, Conditional, complex clauses |
| Vocabulary | ~1000 words | ~2500 words |
| Independence | Predictable situations | Unpredictable situations |
| Past tenses | Basic Indefinido/Imperfecto | Combined narration |

## Key Differences: German B1 vs Spanish B1

| Aspect | German B1 | Spanish B1 |
|--------|-----------|------------|
| Subjunctive | Konjunktiv II (würde + Inf, wäre, hätte) | Presente + Imperfecto de Subjuntivo |
| Conditional | Würde + Infinitiv | Condicional Simple |
| Past narration | Perfekt + Präteritum | Indefinido + Imperfecto |
| Relative clauses | der/die/das + cases | que/quien/donde (simpler) |
| Passive | werden + Partizip II | ser/estar + participio |
| Word order | V2, verb-final in subclauses | More flexible (SVO) |
| Cases | Nominative, Accusative, Dative, Genitive | No case system (prepositions) |
